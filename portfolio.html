RSS Reader

Github Repository
A basic RSS reader made using C/C++, primarily for my own learning and use.

This one was a good challenge. I wanted to make this one primarily to see how difficult it was to do website interactions in C/C++. The plan was to create the entire thing using as few non-standard or third-party libraries as possible. I was able to do it with only one external library (OpenSSL). In theory it works both on Windows and Linux, and it should get the most recent RSS feed from xkcd.com

UnmaskAI

Website
This is a project taken on by myself and one friend to empower and monitor responsible AI usage in schools.

This was definitely the most work I have done on a project. We built a webpage as the base for it using PHP, MySQL, and Apache. To go along with this we also created a browser extension with JavaScript. We were able to use PHP as a way to bridge those apps and connect everything to a central database. We were also able to successfully integrate the Canvas LMS API to enable connections with courses that schools already have.
